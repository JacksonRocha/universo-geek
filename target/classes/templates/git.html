🔄 Rebase da Branch develop em uma HU

Atualize sua branch com:

git checkout feature/TS1503S-1234
git fetch origin
git rebase origin/develop

Se houver conflitos:

git add arquivo-conflitado
git rebase --continue

🔀 Squash - Juntar Commits

Para consolidar commits:

git rebase -i HEAD~n

Mantenha pick no primeiro, e troque para squash nos outros.

📝 Renomear uma Branch

Para renomear:

git branch -m novo-nome

Atualizar no repositório remoto:

git push origin novo-nome
git push origin --delete nome-antigo
git push --set-upstream origin novo-nome

🔍 Ver Logs de Commits

git log --oneline --graph --all

🔢 Ver Alterações da Branch

git diff develop..HEAD
git diff --name-only develop..HEAD
